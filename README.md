##AngularFire Tic Tac Toe

###Basic Two-Player Tic Tac Toe with Firebase providing real-time connection between remote users.

####Technologies Used 
* AngularJS 1.2.6
* jQuery
* Firebase
* AngularFire
* Bootstrap

####Installation Instructions
- Download repo
- Open 'index.html'
- Wait for other players if there are none
- enojy!
